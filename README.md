# Student Skill Analyzer

**Student Skill Analyzer** is a web-based application that helps students track their learning progress and receive personalized career recommendations based on their skills, study hours, and interests.

---

## Features

- **Daily Study Logging**: Record the skills learned, subjects studied, hours spent, and interest level.  
- **Skill Analysis**: Analyze student performance by calculating scores based on study hours and interest.  
- **Personalized Recommendations**: Suggest career paths and next steps tailored to the studentâ€™s strengths.  
- **Dashboard**: Clean and intuitive interface showing skill progress and recommendations.  
- **Motivation & Progress Tracking**: Encourages consistent study habits and self-awareness.

---

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: Python, Flask  
- **Data Storage**: CSV (or Supabase for production)  
- **Libraries**: Pandas (for data analysis)
